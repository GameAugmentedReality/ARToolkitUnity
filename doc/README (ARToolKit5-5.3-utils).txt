Running the utilities
---------------------

ARToolKit includes a set of utilities to perform essential functions required by ARToolKit. After installation, the executables for these utilities can be found in the bin directory inside your ARToolKit directory.

  Windows:

	Each utility can be opened by double-clicking its icon in the ARToolKit5\bin directory. Alternately, you can run it from the command line:

	* Open a command-line window (cmd.exe).
	* Navigate to your ARToolKit5\bin directory.
	* Type (for example): mk_patt.exe

  Mac OS X:
  
    The utilities should be launched from a Terminal window:
  
    * Open a Terminal window (/Applications/Utilties/Terminal).
	* Navigate to your ARToolKit5\bin directory.
	* Type (for example): ./mk_patt.exe


Which utilities are included:
-----------------------------

ARToolKit utilities:
 * calib_camera
 * mk_patt
 * calib_optical
 * genMarkerSet

ARToolKit NFT utilities:
 * genTexData
 * dispImageSet
 * dispFeatureSet
 * checkResolution
 
Required patterns for the utilities, which you can print out, are found in the folder "patterns" inside the "doc" folder. Images for sample NFT markers are in the folder "Marker images" inside the "doc" folder.
